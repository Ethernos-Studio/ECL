# ECL 简介

## 什么是 ECL？

ECL (EthernosCommandLang) 是一个基于 Rust 开发的半过程式编程语言，专为教学和基本数学运算而设计。它具有简洁的语法和直观的概念，非常适合编程初学者学习编程基础。

## 设计理念

ECL 的设计遵循以下几个核心原则：

1. **教学友好**：语法简洁明了，易于理解和学习
2. **功能完整**：包含变量、函数、控制流等基本编程概念
3. **类型安全**：提供静态类型声明和运行时类型检查
4. **错误友好**：提供详细的错误信息和修复建议
5. **交互性强**：内置 REPL 环境，支持实时代码执行和调试

## 核心特性

### 自定义表达式语法
ECL 支持独特的 `expr` 关键字定义表达式函数，允许中缀调用语法，使代码更接近数学表达式的自然写法。

### 多种参数类型
支持多种参数模式，包括 `l`, `r`, `l r`, `r1 r2`, `r1 argv` 等，为表达式函数提供灵活的参数定义方式。

### 类型系统
ECL 提供了完整的类型系统，包括：
- 基本类型：int, float, double, str, bool
- 复合类型：array（数组）, list（列表）
- 静态类型声明和显式类型转换

### 数据结构
- **数组**：固定长度的同类型数据集合
- **列表**：动态长度的异类型数据集合

### 控制流
- 条件语句：if/else
- 循环语句：for/in, while
- 范围表达式：支持 1..5 等范围语法

### 错误处理
详细的错误处理机制，包括：
- 词法错误提示
- 语法错误定位
- 运行时错误追踪
- 友好的错误消息和修复建议

## 应用场景

ECL 主要适用于以下场景：
1. **编程教学**：帮助初学者理解编程基本概念
2. **算法演示**：简洁的语法适合算法演示和教学
3. **数学计算**：支持基本数学运算和表达式计算
4. **原型开发**：快速实现简单的程序原型

## 技术栈

- **语言**：Rust (Edition 2024)
- **构建工具**：Cargo
- **依赖**：无外部依赖，纯 Rust 标准库实现
